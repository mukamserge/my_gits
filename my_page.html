A PROPOS DE GITHUB

Dans ce chapitre, il est question de se familiariser avant les commandes GitHub, en vue de gérer des dépots distants. Nous listons ci-dessous quelques commandes usuelles:

  • Pour Accéder à un dépot distant, taper la commande "git clone...suivi de l'adresse url de la page GitHub de projet où le dépot doit etre effetué"
  • Saisir la commande git "remote add OC suis de la même adresse url précédente"
  • Mettre à jour le dépot local en utilisant la commande "git pull origin main"

    • Celà vous permettra, une fois télécharger de mettre à jour le dépot local en tenant compte des modifications effectuées par les autres collaborateurs.
